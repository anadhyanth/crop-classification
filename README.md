# Crop Classification

A Machine Learning-based Crop Classification system that predicts suitable crop categories using environmental and soil parameters such as Nitrogen, Phosphorus, Potassium, temperature, humidity, pH, and rainfall. Built with Python and Scikit-learn.
# Crop Classification Using Machine Learning

## Project Overview

Crop Classification is a Machine Learning project that predicts the most suitable crop based on various soil and environmental parameters. The system analyzes agricultural data such as Nitrogen, Phosphorus, Potassium, temperature, humidity, pH value, and rainfall to classify crops accurately.

This project demonstrates the application of Machine Learning techniques in smart agriculture, helping farmers and researchers make data-driven decisions for crop selection.

---

## Objectives

* Predict suitable crop categories using agricultural data.
* Improve crop selection accuracy through Machine Learning.
* Demonstrate data preprocessing, model training, and evaluation workflows.
* Provide a foundation for future smart farming applications.

## Dataset Features

The dataset contains the following features:

| Feature        | Description                |
| -------------- | -------------------------- |
| Nitrogen (N)   | Nitrogen content in soil   |
| Phosphorus (P) | Phosphorus content in soil |
| Potassium (K)  | Potassium content in soil  |
| Temperature    | Temperature in °C          |
| Humidity       | Relative humidity (%)      |
| pH             | Soil pH value              |
| Rainfall       | Rainfall in mm             |

### Target Variable

* Crop Type
